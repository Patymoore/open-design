# Deep Research

Use this skill for questions that need more than one search query: market maps, competitor research, technical due diligence, product decisions, policy or standard comparisons, or "find out what is really true" tasks.

## Direct Run

```bash
node --experimental-strip-types scripts/deep-research.ts "topic or question" --rounds 3 --per-query 5
```

The script gathers a deduped research packet. The agent must still open/read the strongest sources and synthesize the answer.

## Agent Workflow

1. Restate the research question and decide the decision the user is trying to make.
2. Generate query families: official/source-of-truth, implementation/examples, criticism/risks, and recent updates.
3. Run `scripts/deep-research.ts` or run `search` manually for each query.
4. Open the most credible primary sources before synthesizing.
5. Produce a cited answer with confidence, contradictions, and recommended next steps.

Do not present search snippets as final truth. Treat them as leads.
