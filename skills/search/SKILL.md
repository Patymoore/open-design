# Search

Use this skill when the user asks to search, verify, compare current options, gather links, inspect documentation, or answer anything that may have changed recently.

## Direct Run

From the staged skill root:

```bash
node --experimental-strip-types scripts/search.ts "your search query" --limit 8
```

Behavior:

- If `BRAVE_API_KEY` is set, the script calls Brave Search.
- Without `BRAVE_API_KEY`, it uses DuckDuckGo HTML search with no required key.
- Use `--json` when the agent needs machine-readable results.

## Agent Workflow

1. Rewrite the user's request into 2-4 precise search queries when the topic is broad.
2. Prefer primary sources: official docs, source repos, release notes, standards, laws, product pages, or direct authors.
3. Run the script for each query, dedupe URLs, then open/read the strongest sources before answering.
4. Cite links inline and distinguish facts found in sources from agent inference.
5. For complex research, chain this skill with `deep-research` or `last30days`.
